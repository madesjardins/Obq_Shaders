Obq_MeasuredMaterial


Class

Material


Output

Color RGB


Synopsis

Obq_MeasuredMaterial is a shader that reads binary BRDF files (MERL database .binary or MaterialSketcher .mskbrdf).

Warning : This shader is in development right now and not all the parameters are active.


Fig.1 : In this example, the Stanford dragon is shaded using the MERL red metallic paint BRDF.


Usage

Simply browse for a binary BRDF file and render.


BRDF


Filename [filename]

The binary BRDF filename, either MERL database or MaterialSketcher (not yet public) BRDF.


Resolution [resolution]

The LUT resolution for that BRDF inside the shader.


Direct


Color [direct_color]

A direct color multiplier.


Scale [direct_scale]

A direct scale factor.


Use MIS (test) [useMIS]

Use multiple importance sampling for direct illumination.


Inirect


Color [indirect_color]

An indirect color multiplier.


Scale [indirect_scale]

A indirect scale factor.


Use MIS (test) [useImportanceSampling]

Use multiple importance sampling for indirect illumination.


Samples [samples]

The square root of the number of samples to use for indirect illumination.


Options


Stop binary search when in range of [binMaxDif]

An optimization for the MIS and IS.


Use jitters [useJitters]

Use a bit of randomness to break repeating patterns.


Use half hints (test) [useHalfs]

Another optimization for the MIS and IS.


AOVs


Direct [direct_AOV]

The direct AOV.


Indirect [direct_AOV]

The indirect AOV.
