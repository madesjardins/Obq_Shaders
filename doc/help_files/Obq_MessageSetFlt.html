Obq_MessageSetFlt


Class

Texture [Passthrough > Message > Set]


Output

Color RGBA


Synopsis

Attaches a scalar value to a key in the message bank. You can use the Obq_Message nodes to create reflection mattes, use different images, etc. depending on what object is hit first and stuff like that. These nodes are basically simplified version of Stefano's sja_radio shaders.


Usage

Use an Obq_MessageSetFlt as a passthrough to attach a value to a key, pass it downstream and use an Obq_MessageGetFlt to fetch it. You can also use a Obq_MessageStoreFlt to fetch and store that value in a AOV, this is useful for saving reflection matte.




Passthrough [passthrough]

The passthrough port for the shader.


Key [key]

The key (string).


Value [value]

The value to attach to the key.


Set value after passthrough evaluation [setAfter]

Instead of attaching the value before evaluation, attach it after.
