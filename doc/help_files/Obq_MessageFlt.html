Obq_MessageFlt


Class

Texture


Output

Scalar


Synopsis

Outputs the scalar value attached to a message in the message bank.


Fig.1 : Obq_Toon writes the scalar input value the gradient should have into the message "OToon" and Obq_MessageFlt gets it.

Note : At the moment, only the Obq_Toon with the gradient workflow needs this shader. For a more complete set of such functions, take a look at Stephano's ssj_radio collection shaders.


Usage

Simply connect this into a scalar port and type the message's name or choose a preset to fetch the associated scalar value in that ShaderGlobals. If the message doesn't exist, the Default Value will be returned




Default Value [defaultValue]

If a message doesn't exist, this value will be output.


Mode [mode]
•Custom:use a custom message name.
•Preset:use a predefined message name from a list of presets.

Preset [preset]

Available when you choose Preset mode, this is a list of predefined messages.


Message [message]

The name of the message to fetch.
