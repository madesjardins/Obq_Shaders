Obq_TransformShadingPoint


Class

Texture [Passthrough > Modify > Shader Globals]


Output

Color RGBA


Synopsis

Obq_TransformShadingPoint is a passthrough node that transforms the shading point. This is particiularly useful when you want to move VDB simulations coming from Houdini.


Usage

Use as the node that connects into the material node (surface or volume port).


Passthrough


Shader [passthrough]

The input for your shading branch.


Coordinates


Space [coordinates]

The space coordinates for this modification :

  • Object
  • World

Transforms

Use the buttons to link or set to objects transform.


Transform mode [mode]

Options are :

  • SRT
  • Matrix

Invert [invert]

This inverts the transforms.


Rotation Order [rotationOrder]

This is the rotation order needed for the SRT mode.
